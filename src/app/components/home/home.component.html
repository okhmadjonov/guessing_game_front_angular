Welcome to the number guessing game!

Before starting the game, let me introduce the game rules !!!
1. To start the game you need to register first. ( After that the game would be eligible to start. )
2. Then go to the start game menu, there click the button to generate the session ID.
3. Below there are two fields to insert session ID and guess number.
4. Then, generated session ID would place automaticaly in session ID fields and write your guessed number.
5. Click guess button, then you will see your result win or lose, according to result you will continue.
Hope, you understand how to play the game, good luck !!!
